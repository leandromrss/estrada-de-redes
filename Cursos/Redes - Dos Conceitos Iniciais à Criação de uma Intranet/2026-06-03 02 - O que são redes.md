## Conceitos Aprendidos

- Nossos dispositivos são como nós que funcionam para prover a conexão entre dispositivos
- A rede pode possuir diversas conexões entre os nós, para que, caso um se rompa, a comunicação não se perca.
- Para a comunicação funcionar, todos os dispositivos precisam estar conectado em um mesmo padrão.

---

## Anotações

- A internet é uma rede global de dispositivos
- Precisa de mais de um tipo de protocolo para poder realizar a comunicação entre os dispositivos

---

## Aplicação Prática

- Para nos comunicarmos com outra pessoa, seguimos alguns padrões de comunicação. Por exemplo: "oi, bom dia, tudo bem?" Quando fazemos essa pergunta, esperamos a pessoa responder para seguirmos a conversa baseada na resposta e no padrão de resposta da pessoa. Assim também são os dispositivos dentro de uma rede

## Resumo da Aula

- A rede de computadores, nada mais é que uma comunicação entre dispositivos. Para a comunicação entre esses dispositivos funcionar, é necessário seguir um padrão de configuração ou protocolos. Caso um dos dispositivos da rede esteja fora desse padrão, ele não se comunica com os demais.
